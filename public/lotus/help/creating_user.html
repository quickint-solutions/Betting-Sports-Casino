Creating a New Admin Account

This section describes how to create a new user for the Company. a Customer.

To create a new user, follow this procedure:
Navigate to the Admin Users > User Listing menu item.

Click the Create New User button.

Enter a username, a password and confirm the password. Also, set the user status to Active or Inactive.

Note: Inactive users cannot log in to the site.
A username must be at least four characters consisting of lower case letters, numbers and dots ( . ). Other special characters are not allowed.
Passwords must be between 6 and 15 characters and must contain at least one number and one letter. Special characters are not allowed.
Entering a first or last name is optional.
Click the Create button to create the new user.
Important: When the user is created, the system does not set up their access rights automatically. You have to set up the access rights manually for each user that you have created.
After the user is created, click the icon to set up its access rights.
